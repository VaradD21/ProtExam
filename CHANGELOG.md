# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-03-26

### Added
- Enhanced login page with improved UX
- Better error handling and validation
- Security headers for attack prevention
- Comprehensive documentation

### Fixed
- Frontend-backend connection issues
- DOM loading timing issues
- Form validation errors

### Changed
- Port configuration from 5001 to 8000
- API error messages to be more descriptive
- Improved logging throughout the application

### Security
- Added CORS security headers
- Implemented X-Frame-Options protection
- Added X-XSS-Protection headers
- Improved password handling

## [1.0.0] - 2024-03-20

### Added
- Initial release
- Dual role system (Student/Organizer)
- Exam management functionality
- Anti-cheating detection system
- Real-time monitoring via WebSocket
- Auto-grading for MCQ questions
- Student exam interface with timer
- Organizer dashboard for monitoring
- Comprehensive logging system
- Token-based authentication
